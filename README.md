### Lab Objectives
*	Secure Remote Access: Implement Multi-Factor Authentication (MFA) to protect RDP sessions from credential-based attacks.
*	Zero Trust Integration: Validate user identity via the Duo Mobile app before granting server access.
*	Infrastructure Hardening: Ensure compliance with security best practices for Windows Server environments.
